# **App Name**: Creature Forge

## Core Features:

- Main Menu: Display a main menu with options to start crafting creatures, view the gallery, and access settings.
- Crafting Interface: Provide an interface where users can select different parts to build their creature.
- Creature Gallery: Show a gallery of created creatures with details.
- Settings: Implement a settings panel with options for the user, like to control sound volume.

## Style Guidelines:

- Primary color: Gold (#FFD700) to evoke a sense of creation and preciousness.
- Background color: Dark gray (#333333) with a metallic sheen, for an industrial fabrication feel.
- Accent color: Silver (#C0C0C0) for highlights and interactive elements to enhance the metallic theme.
- Body and headline font: 'Space Grotesk' for a futuristic and technological appearance.
- Use gear and cog icons to represent the fabrication theme.
- Design the main menu with a central fabrication unit focus, options around.
- Animate metallic reflections on main elements. Slight movements to simulate operation.
